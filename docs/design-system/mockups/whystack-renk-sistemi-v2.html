5 · Hat Renkleri YENİ — 8 bölüm, B1-B8

Kritik ilke: yeşil sahnede yeşil hat olmaz. Zemin yeşil olduğu için hat paletinden yeşil ailesi tamamen çıkarıldı (Success ile de çakışmasın). Altın = B1 ana hat; aynı zamanda "aktif/senin hattın" dili. 8 renk, koyu zeminde eş parlaklıkta seçildi; renk körlüğü için hat adı etiketi + durak numarası her zaman renkle birlikte gösterilir:

KULLANIM HİYERARŞİSİ
ALTIN = "SEN"

Konumun, aktif hattın, CTA'lar, ilerleme. Altın her zaman kullanıcının kendisini işaret eder.

HAT RENKLERİ = YAPI

Haritada çizgiler, kapsam parantezleri, konu etiketleri. Yapıyı anlatır, asla buton olmaz.

SEMANTİK = GERİ BİLDİRİM

Doğru/yanlış/uyarı/bilgi. Anlıktır, geçicidir. B6 kırmızısı ile Error aynı ton — kasıtlı: ikisi de "dur, dikkat" der.

Pasif hat kuralı: Açılmamış ("yakında") hatlar kendi renklerinin %16-20 opaklığında çizilir — renkli ağ vaadi görünür ama sahneyi işgal etmez.

6 · Syntax Teması — "Basamak Dark" YENİ — markalaşma fırsatı

Mockuplarda VS Code'un varsayılan renklerini ödünç almıştım (mor keyword, camgöbeği tip) — palete yabancı duruyorlar. Öneri: kod renkleri de sistemden türesin. Fonksiyon = altın (yine "aksiyon = altın" dili), keyword = adaçayı, tip = camgöbeği, string = bakır. Bu tema ileride VS Code / Rider eklentisi olarak yayınlanabilir — bedava marka yüzeyi:

BASAMAK DARK — ÖRNEK
// Basamak Dark syntax teması
public async Task<Order> GetOrderAsync(int id)
{
    var order = await _http
        .GetFromJsonAsync<Order>($"/orders/{id}");
    return order;
}
